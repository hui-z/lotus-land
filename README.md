# 幻想乡 Lotus Land

在幻想乡里，
每个~~中二~~程序员都会给其他程序员分享各种东西。

本项目就是记录这些程序员究竟讲了什么。


## 规约

* 分享频率：每周一次
* 分享时间：每周三晚上7点
* 分享地点：<凯利大厦2楼>会议室
* 分享人：由本周的分享者指定下一个分享者
* 原则上五次分享内，分享人不能重复
* 欢迎自荐分享（提PR）
